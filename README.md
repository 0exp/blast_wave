A set of useful **rack** middlewares.

## Installation

```ruby
gem 'blast_wave'
```

```shell
bundle install
```

```ruby
require 'blast_wave'
```

- [Rack::BlastWave::RequestId](#rackblastwaverequestid) - provides an unique id attribute to the each request;
